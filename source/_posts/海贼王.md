#### 剧情介绍


{% note info::路飞吵闹着登红发船，误吃了橡胶果实，山贼王前来酒吧捣乱，~~爆杀红发，酒瓶砸头~~，因为香克斯没理会山贼王，导致路飞很生气。 %}
{% note info::几天后，路飞在酒吧为红发说理，遭到山贼王绑架，香克斯从山贼西格手中救下路飞，同时被海王类咬断一直是，并且跟他约好了，在路飞成为最强悍的海贼之后，必须把草帽还给他。 %}
{% note info::十年后，路飞长大了，独自出航寻找伙伴。 %}
